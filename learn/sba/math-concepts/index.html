1. Reading Decimals (5.NBT.3a)
Place Value
3.142 = 3 + 0.1 + 0.04 + 0.002
Each digit has a place value. The decimal point separates the whole number from the fractional part. Read the decimal as "and" when speaking: 3.142 = "three and one hundred forty-two thousandths."
Worked Example
Write 4.075 in words → "four and seventy-five thousandths"
4 = ones, 0 = tenths, 7 = hundredths, 5 = thousandths
2. Multiply Whole Numbers (5.NBT.5)
Standard Algorithm
2,684 × 24 = 64,416
Multiply by each digit separately, shifting left for tens. Then add all partial products together. Always line up digits by place value.
Worked Example
346 × 27:
346 × 7 = 2,422
346 × 20 = 6,920
Total = 9,342
3. Add/Subtract Fractions (5.NF.1)
Common Denominators
a/b + c/d = (ad + bc) / bd
To add or subtract fractions, they must have the same denominator (bottom number). Find the Least Common Denominator (LCD), then convert both fractions before adding.
Worked Example
1/3 + 1/4 → LCD = 12
= 4/12 + 3/12 = 7/12
4. Fraction = Division (5.NF.3)
Key Relationship
a ÷ b = a/b
Any fraction can be read as a division problem. The numerator (top) is divided by the denominator (bottom). This helps understand what fractions mean in real life.
Worked Example
5 sandwiches shared by 6 people:
5 ÷ 6 = 5/6 — each person gets 5/6 of a sandwich
5. Patterns (5.OA.3)
Pattern Relationship
If Rule A gives x, Rule B gives 2x
Two patterns can be generated from different rules starting at the same point. Compare them to find the relationship. Rule B always produces numbers twice as large as Rule A.
Worked Example
Rule A (+3): 0, 3, 6, 9, 12…
Rule B (+6): 0, 6, 12, 18, 24…
Each Rule B term = 2 × Rule A term
6. Plot Coordinates (5.G.1)
Coordinate Rules
Point (x, y): x = right, y = up
In the first quadrant, both x and y are positive. Always start at (0,0), go right x units, then go up y units. The x-axis is horizontal; the y-axis is vertical.
Worked Example
Plot (3, 4): go right 3, then up 4
Plot (0, 2): stay at x=0, go up 2
🟡 Medium
7. Multiply Fractions (5.NF.4)
Fraction Multiplication
a/b × c/d = ac / bd
Multiply numerators together and denominators together. Use an area model: divide a rectangle into rows for one fraction and columns for the other. The shaded overlap is the product.
Worked Example
2/3 × 3/4 = (2×3)/(3×4) = 6/12 = 1/2
8. Compare Fractions by Scaling (5.NF.5a)
Scaling Rule
If n > 1, then a × n > a
If n < 1, then a × n < a
Multiplying by a fraction less than 1 makes the result smaller. Multiplying by a fraction greater than 1 makes it larger. Use this to predict whether a product is bigger or smaller without calculating.
Worked Example
8 × 3/4 = 6 (smaller than 8, because 3/4 < 1)
8 × 5/4 = 10 (larger than 8, because 5/4 > 1)
9. Divide Unit Fractions (5.NF.7)
Unit Fraction Division
a ÷ (1/n) = a × n
To divide by a unit fraction, multiply by its reciprocal. Think: "How many 1/4s fit inside 3?" Each whole has 4 quarters, so 3 wholes have 12 quarters.
Worked Example
3 ÷ 1/4 = 3 × 4 = 12
1/4 ÷ 3 = 1/4 × 1/3 = 1/12
10. Volume Formula (5.MD.5)
Volume Formulas
V = l × w × h
or V = B × h (B = base area)
Volume measures the space inside a 3D shape in cubic units. Multiply length × width to get the base area (B), then multiply by height to stack the layers.
Worked Example
Box 7 × 2 × 4:
V = 7 × 2 × 4 = 56 cubic units
11. Line Plots with Fractions (5.MD.2)
Line Plot Rule
total = sum of all data values
A line plot displays data along a number line. Each X mark represents one data point. Stack X marks above a value when multiple data points share the same value. Use fractions for measurements.
Worked Example
3 students measured 1/2 cup, 2 measured 3/4 cup.
Plot 3 X's above 1/2 and 2 X's above 3/4
12. Powers of 10 (5.NBT.2)
Place Value Shift
n × 10ᵖ shifts decimal p places right
Each power of 10 moves the decimal point one place to the right when multiplying (or left when dividing). The exponent tells you how many zeros follow the 1.
Worked Example
4.5 × 10³ = 4,500 (move decimal 3 right)
4.5 ÷ 10² = 0.045 (move decimal 2 left)
🔴 Hard
13. Decimal Operations (5.NBT.7)
Decimal Operations
+/−: Line up decimal points
×: Count total decimal places
For addition and subtraction, always line up decimal points before computing. For multiplication, count the total decimal places in both factors — the product has that many decimal places.
Worked Example
2.4 × 1.5: 24 × 15 = 360, then 2 decimal places → 3.60
3.75 + 1.5 = 5.25 (line up decimal points)
14. Real-World Fraction Multiplication (5.NF.6)
Mixed Number Multiplication
Convert to improper fractions first
5/2 × 4/3 = 20/6 = 3⅓
To multiply mixed numbers, first convert each to an improper fraction. Then multiply numerators together and denominators together. Simplify the result back to a mixed number if needed.
Worked Example
Garden 2½ ft × 1⅓ ft:
5/2 × 4/3 = 20/6 = 3⅓ sq ft
15. Classify 2D Shapes (5.G.3/4)
Shape Hierarchy
A square is always a rectangle
A rectangle is sometimes a square
Shapes fit in a hierarchy. A more specific shape inherits all properties of the broader category. Every square has 4 right angles AND 4 equal sides, so it qualifies as both a rectangle and a rhombus.
Worked Example
"Is a square always a rectangle?" YES
All 4 angles are 90° — that's what makes it a rectangle.
"Is a rectangle always a square?" NO — sides may be unequal.
16. Composite Volume (5.MD.5c)
Composite Volume
V_total = V₁ + V₂
For irregular 3D shapes, split them into rectangular prisms (boxes) that don't overlap. Calculate each box's volume separately, then add them together for the total volume.
Worked Example
Box 1: 3×3×4 = 36 cubic units
Box 2: 2×2×3 = 12 cubic units
Total: 36 + 12 = 48 cubic units
17. Order of Operations (5.OA.1/2)
Order of Operations
( ) → × ÷ → + −
Parentheses → Multiply/Divide → Add/Subtract
Always work inside parentheses first. Then do multiplication and division left to right. Finally do addition and subtraction left to right. Parentheses change the order and the answer!
Worked Example
(8+4) × 3 = 12 × 3 = 36
8 + 4 × 3 = 8 + 12 = 20
18. Measurement Conversion (5.MD.1)
Conversion Rule
Multiply to go smaller unit
Divide to go larger unit
To convert to a smaller unit (like km to m), multiply. To convert to a larger unit (like cm to m), divide. Always multiply by the conversion factor that links the two units.
Worked Example
3.5 kg → g: 3.5 × 1000 = 3,500 g
250 cm → m: 250 ÷ 100 = 2.5 m
