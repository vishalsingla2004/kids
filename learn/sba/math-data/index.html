1. Reading Bar Graphs
Reading Rule
Read top of bar → find value on y-axis
Bar graphs use bar height to show quantity. The y-axis (vertical) shows the scale. Find the top of each bar and trace left to the y-axis to read its value. The tallest bar shows the most.
Worked Example
"Which month had most rain?" → Find the tallest bar → March (7 inches)
2. Frequency Tables
Frequency Rule
Count tallies for each category
A frequency table organizes data by category and count. Tally marks group counts in fives (four vertical + one diagonal). Count up all tallies in each row to get the frequency.
Worked Example
Pets: IIII = 4 dogs, IIII I = 5 cats, II = 2 fish
Total: 4 + 5 + 2 = 11 pets
3. Number Lines
Number Line Rule
Divide segment into equal parts
To place fractions on a number line, divide the distance between whole numbers into equal parts matching the denominator. The numerator tells you which tick mark to use.
Worked Example
Place 3/4: split 0-1 into 4 equal parts, mark the 3rd tick
1.5 is halfway between 1 and 2
4. Mean (Average)
Mean Formula
mean = sum ÷ count
The mean (average) is found by adding all values and dividing by how many values there are. It represents the "fair share" if all values were evened out. Think of leveling out stacks of blocks.
Worked Example
Scores: 80, 90, 70, 100
Sum = 340, Count = 4
Mean = 340 ÷ 4 = 85
5. Pictographs
Pictograph Formula
Count symbols × key value = total
A pictograph uses symbols to represent data. The key tells you what each symbol equals. Multiply the number of symbols by the key value. Half symbols represent half the key value.
Worked Example
Mon: 3 suns × 2 = 6 sunny days
Wed: 3½ suns × 2 = 3×2 + 1 = 7 sunny days
6. Dot Plots
Dot Plot Rule
Each dot = one data point; mode = tallest stack
Dot plots show each individual data value as a dot. Stack dots above a value when multiple data points share it. The tallest stack shows the mode (most common value).
Worked Example
Value 3 has 3 dots (tallest stack) → mode = 3
Total data points = count all dots
🟡 Medium
7. Line Plots with Fractions (5.MD.2)
Line Plot Rule
Mark each data value with X above number
Fractional line plots are used for measurement data (like lengths). Place an X above the number line at each measured value. Use fractions for the tick marks based on the smallest measurement unit.
Worked Example
Heights measured in 1/4-inch: mark X above each value.
3 X's above 1/2 means 3 students measured exactly 1/2 inch
8. Writing Equations from Patterns
Equation from Table
Find the rule: y = mx
Look at how y changes as x increases. If y = x × constant, the equation is y = mx. Check: does the rule work for every row? If yes, you've found the equation for the pattern.
Worked Example
Pattern: multiply by 3 each time.
Equation: y = 3x
Check: x=4 → y = 3×4 = 12 ✓
9. Data Story Problems
Data Story Strategy
Read all values → answer specific question
When solving data story problems, first read all values carefully. Identify what the question is asking (total, difference, most, least). Then perform the correct operation on the right values.
Worked Example
"How many more chose blue vs. red?"
Blue = 10, Red = 7
10 − 7 = 3 more chose blue
10. Comparing Two Data Sets
Comparison Method
Compare means, ranges, modes
To compare two data sets, calculate the mean of each. A higher mean means a higher typical value. Also compare ranges (spread) and modes. Two data sets can look different but have similar means.
Worked Example
Class A mean = 72, Class B mean = 81
Conclusion: Class B scored higher overall by 9 points on average
11. Plotting Ordered Pairs (5.G.1)
Coordinate Rule
(x, y) → right x, up y
Graph data as coordinate points to see patterns visually. If points form a straight line, there's a constant relationship between x and y. The pattern shows a proportional relationship.
Worked Example
Time vs. distance: (1,5), (2,10), (3,15)
Plot each point → they form a straight line (proportional!)
12. Sum/Difference from Graphs
Graph Calculation
Add bar segments for total; subtract for difference
Stacked bar charts show two categories per bar. Add the segments to find the total for that group. Subtract corresponding bars to find differences between groups.
Worked Example
Jan = 16 + 8 = 24, Feb = 13 + 7 = 20
Total for two months = 24 + 20 = 44
Difference = 24 − 20 = 4

13. Multi-Step Fraction Line Plots
Line Plot Calculations
Sum all values; find differences
To find the total from a line plot, add up all data values, multiplying each value by the number of X marks above it. To find differences, subtract one sum from another.
Worked Example
2×(1/4) + 3×(1/2) + 1×(3/4) + 2×(1) =
1/2 + 3/2 + 3/4 + 2 = 19/4 = 4¾ yards total
14. Variable Equations from Data (5.OA)
Data Equation
known + x = total → x = total − known
When data shows a total and some known values with a missing piece, write an equation. Isolate the variable by using the inverse operation (if subtracted, add; if added, subtract).
Worked Example
p − 8/5 = 35/7
p = 35/7 + 8/5 = 175/35 + 56/35 = 231/35 = 33/5
15. Interpreting Trends
Trend Analysis
Increasing pattern → positive relationship
A trend shows the general direction data moves. An upward trend means as x increases, y increases (positive relationship). Use the trend to make predictions beyond the given data points.
Worked Example
More study hours → higher scores (upward trend).
Based on the trend: 5 study hours would predict a high score ≈ 95+
16. Statistical Questions
Statistical Question
Expects variability in answers
A statistical question anticipates answers that vary across individuals. Non-statistical questions have a single definite answer. Statistical questions are the kind that can be answered by collecting data from a group.
Worked Example
"How old is my teacher?" ✗ (one answer)
"How old are students in our class?" ✓ (varied)
"What is 2 + 2?" ✗ vs "How many pets do students have?" ✓
17. Data-Supported Arguments
Data Argument
Claim → Data Evidence → Conclusion
When making claims from data, always cite specific numbers. State your claim, point to the specific data that supports it (exact percentage or value), and explain what it means in context.
Worked Example
Claim: "Most students prefer math"
Evidence: pie chart shows 60% chose math
Conclusion: 60% > 50%, so math is the majority preference ✓
18. Complex Comparison Problems
Unit Price Comparison
Divide total by quantity for each option
To compare value, find the unit rate (price per item, cost per ounce, etc.) for each option. Calculate by dividing total cost by quantity. The lower unit rate gives the better value.
Worked Example
Option A: $12 ÷ 4 = $3.00/item
Option B: $15 ÷ 6 = $2.50/item
Option B is cheaper per item
